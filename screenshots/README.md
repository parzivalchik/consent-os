# Screenshots

Place the following screenshots here:

1. `dashboard-dark.png` - Dashboard in dark mode (full page)
2. `dashboard-light.png` - Dashboard in light mode (full page)  
3. `popup-dark.png` - Extension popup in dark mode
4. `popup-light.png` - Extension popup in light mode

To capture screenshots:
1. Open the dashboard/popup in Chrome
2. Use Cmd+Shift+4 (Mac) or Win+Shift+S (Windows) to capture
3. Save to this folder with the appropriate filename
